Word Counter App

Informazione: Word Counter App è un'applicazione che permette di analizzare testi e file per contare il numero totale di parole, lettere, spazi e identificare parole ripetute.

Descrizione del Progetto

Word Counter App è un'applicazione sviluppata per fornire un'analisi dettagliata di testi e file di testo. L'applicazione conta il numero di parole, lettere, spazi e identifica le parole ripetute in un dato input. È utile per analisi linguistiche, statistiche di testo e varie altre applicazioni dove è necessario un conteggio preciso degli elementi di un testo.

Tecnologie Utilizzate

  • Node.js: Un runtime JavaScript asincrono e basato su eventi, progettato per creare applicazioni di rete scalabili. Utilizzato per eseguire il backend dell'applicazione.
  • TypeScript: Un superset di JavaScript che aggiunge tipizzazione statica e altre funzionalità avanzate. Utilizzato per garantire un codice più robusto e manutenibile.
  • Jest: Un framework di testing JavaScript. Utilizzato per scrivere e eseguire test automatici, garantendo che il codice funzioni correttamente e che le modifiche non introducano regressioni.
  • npm (Node Package Manager): Gestore di pacchetti per Node.js. Utilizzato per gestire le dipendenze del progetto.
  • fs (File System): Modulo nativo di Node.js utilizzato per operazioni di lettura e scrittura di file.
  • Docker: Utilizzato per creare un ambiente di sviluppo e produzione isolato. Docker consente di eseguire l'applicazione in un container, assicurando che funzioni nello stesso modo su qualsiasi macchina.

Requisiti

  • Node.js
  • npm (Node Package Manager)
  • Docker

Installazione e Avvio del Progetto

Installazione Locale

  1. Scarica o clona il repository del progetto.
  2. Installa le dipendenze: 
    npm install
  3. Avvia l'applicazione: 
    npm start

Utilizzo di Docker

  1. Costruisci l'immagine Docker: 
    docker build -t word-counter-app .
  2. Esegui il container Docker: 
    docker run -d -p 3000:3000 word-counter-app

Utilizzo dell'Applicazione

Analisi di un Testo

Per analizzare un testo, puoi utilizzare la funzione analyzeText.

Esempio

import { analyzeText } from './src/analyzeText';

const text = 'hello world hello';
const result = analyzeText(text);
console.log(result);

Analisi di un File

Per analizzare un file, puoi utilizzare la funzione analyzeFile.

Esempio

import { analyzeFile } from './src/analyzeFile';

const filePath = './path/to/your/test/file.txt';
analyzeFile(filePath).then(result => {
    console.log(result);
}).catch(error => {
    console.error('Error analyzing file:', error);
});

Test

Per eseguire i test automatici, puoi utilizzare il comando npm test. Questo comando eseguirà i test definiti nella directory tests utilizzando il framework Jest.
